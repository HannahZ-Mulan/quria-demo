UI Optimization · Spec

让 Quria 的 UI 高级、灵动、活起来

当前首页是 shadcn 默认卡片 + 平淡的 hover:shadow-lg，缺乏层次与呼吸感。下面给出 3 个优化方向，均满足「卡片 hover 浮动 + 整体灵动」的核心诉求，气质与复杂度各异。

💡 把鼠标移到下方任意卡片上，即可实时体验每个方案的浮动 / 灵动效果。
方案 A

极光玻璃拟态 · Glassmorphism Aurora

毛玻璃卡片 + 流动的极光渐变背景。hover 时卡片向上浮起、微倾斜，并有一道光泽从左扫到右。整体克制、通透、高级感强，最贴合「灵动」二字。

气质：通透 / 梦幻 / 优雅 配色：深蓝夜空 + 紫青粉极光 复杂度：低（纯 CSS，无 JS） 性能：优 推荐场景：品牌首页、展示型落地页

方案 B

3D 透视跟随 · Tilt Card

鼠标在卡片上移动时，卡片基于鼠标位置实时做 3D 透视倾斜（rotateX / rotateY 跟随光标），同时悬浮、带立体厚度边与跟随鼠标的霓虹辉光。交互最强烈、最有「活」的感觉，科技感拉满。

气质：科技 / 立体 / 未来感 配色：深空黑 + 青蓝霓虹辉光 复杂度：中（CSS + 少量 JS 跟踪鼠标） 性能：良（需限流 mousemove） 推荐场景：产品特性展示、技术 Demo

方案 C

弹性分层浮起 · Spring Parallax

弹性弹簧动效（spring 缓动），hover 时卡片整体上浮，同时标题 / 按钮 / 图标分层错位浮起（视差），元素带轻微缩放与颜色过渡。温暖、活泼、有呼吸感，最贴近现有浅色品牌基调，改动最小。

气质：活泼 / 温暖 / 呼吸感 配色：浅紫渐变 + 紫色强调 复杂度：低（纯 CSS，无 JS） 性能：优 推荐场景：与现有浅色 UI 无缝衔接

三个方案横向对比

同样满足「hover 浮动 + 灵动」，差异在气质、配色与实现成本。

一句话最「仙」最「炫」最「稳」
