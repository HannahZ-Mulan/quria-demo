语言切换下拉框（Language Switcher Dropdown）

本页为 SPEC-001-language-switcher-dropdown.md 的可视化高保真原型，展示组件在各页面、各交互状态、桌面/移动端、以及 RTL（希伯来语）场景下的最终呈现。视觉严格遵循项目 shadcn/ui neutral 令牌与 LanguageSwitcher.tsx 的实现。

语言数：9 触发器：图标 + 语言名 定位：右上角 absolute（随滚动） 组件：LanguageSwitcher.tsx 来源：lucide-react 图标 + Button 基础组件

1组件交互状态

同一个组件的四种关键状态。触发器为胶囊形 outline 按钮，含地球图标 + 当前语言名 + 旋转箭头；展开菜单右对齐、9 项可滚动，当前语言以浅灰底 + 对勾标记。

状态 A · Collapsed（默认）
折叠态。仅显示地球图标 + 当前语言 + 下箭头。点击切换为展开。
状态 B · Expanded（展开）
展开态。9 种语言，当前项 简体中文 高亮 + 对勾；菜单右对齐不溢出视口。
状态 C · Hover / Focus
悬停/聚焦态。触发器显示焦点环；菜单项 Italiano 处于 hover/focus 高亮，对勾仅出现在当前已选 English。
  • 1触发器结构：<Languages icon/> + 当前语言全称 + <ChevronDown/>，胶囊形 outline 按钮（高 34px）。
  • 2当前语言标记：浅灰底 bg-muted + 加粗 + 右侧 Check 对勾，aria-checked="true"。
  • 3关闭方式：点击外部、按 Esc、或选中任一项（含重复选中当前项）。
  • 4无障碍：触发器 aria-haspopup="menu" + aria-expanded；菜单项 role="menuitemradio"。
